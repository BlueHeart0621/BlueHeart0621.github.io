前缀函数

1. 定义

1.1 前缀 & 真前缀

前缀是指从串首开始到某个位置 iii 结束的一个特殊子串。字符串 SSS 的以 iii 结尾的前缀表示为

prefix(S,i)=S[0..i]\begin{array}{c} prefix(S,i) = S[0..i] \end{array} prefix(S,i)=S[0..i]​

真前缀指除了 SSS 本身的 SSS 的前缀。

1.2 后缀 & 真后缀

后缀是指从某个位置 iii 开始到整个串末尾结束的一个特殊子串。字符串 SSS 的从 iii 开头的后缀表示为

suffix(S,i)=S[i..∣S∣−1]\begin{array}{c} suffix(S,i) = S[i..|S|-1] \end{array} suffix(S,i)=S[i..∣S∣−1]​

真后缀指除了 SSS 本身的 SSS 的后缀。

1.3 前缀函数

给定一个长度为 nnn 的字符串 sss，其前缀函数定义为一个长度为 nnn 的数组 π\piπ。其中 π[i]\pi[i]π[i] 含义为：

  • 如果子串 s[0..i]s[0..i]s[0..i] 有相等的真前缀 s[0..kj−1]s[0..k_j-1]s[0..kj​−1] 和真后缀 s[i−(kj−1)..i]s[i-(k_j-1)..i]s[i−(kj​−1)..i]，那么 π\piπ 为最大的相等的真前后缀长度，即

π[i]=max{kj}\begin{array}{c} \pi[i] = max\{k_j\} \end{array} π[i]=max{kj​}​

  • 如果子串 s[0..i]s[0..i]s[0..i] 没有相等的真前后缀，则

π[i]=0\begin{array}{c} \pi[i] = 0 \end{array} π[i]=0​

1.4 字符串的周期

对于字符串 sss 和 0<p≤∣s∣0 \lt p \leq |s|0<p≤∣s∣，若 s[i]=s[i+p] s[i] = s[i+p] s[i]=s[i+p] 对于所有 i∈[0,∣s∣−p−1] i \in [0, |s|-p-1] i∈[0,∣s∣−p−1] 成立，则称 ppp 是 sss 的周期。

1.5 字符串的 border

对于字符串 sss 和 0≤r<∣s∣0 \leq r \lt |s|0≤r<∣s∣，若 sss 长度为 rrr 的前缀和长度为 rrr 的后缀相等，就称 sss 长度为 rrr 的前缀（后缀）是 sss 的 border 。

【注】易知前缀函数 π[i]\pi[i]π[i] 对应的就是字符串 s[0..i]s[0..i]s[0..i] 的最长 border 的长度。

2. 性质

  • 如果字符串 sss 有长度为 rrr 的 border，则 ∣s∣−r|s| - r∣s∣−r 是 sss 的周期。

  • 如果字符串 sss 的前缀函数为 π\piπ，∣s∣=n |s| = n ∣s∣=n，则：

  1. sss 所有的 border 长度为 π[n−1],π[π[n−1]−1],⋯ \pi[n-1],\pi[\pi[n-1]-1],\cdots π[n−1],π[π[n−1]−1],⋯ 。
  2. sss 所有的周期为 n−π[n−1],n−π[π[n−1]−1],⋯ n-\pi[n-1],n-\pi[\pi[n-1]-1],\cdots n−π[n−1],n−π[π[n−1]−1],⋯ 。
  3. π[n−1]\pi[n-1]π[n−1] 是 sss 的最长 border 的长度，n−π[n−1]n - \pi[n-1]n−π[n−1] 是 sss 的最小周期。

3. 实现

根据前缀函数的定义我们可以发现，相邻的前缀函数值至多增加 1 ，故可以得到字符串 sss 的前缀函数的计算公式：

  • s[0]=0s[0] = 0s[0]=0 。

  • 如果 s[i]=s[π[i−1]]s[i] = s[\pi[i-1]]s[i]=s[π[i−1]]，则

π[i]=π[i−1]+1\begin{array}{c} \pi[i] = \pi[i-1] + 1 \end{array} π[i]=π[i−1]+1​

  • 如果 s[i]≠s[π[i−1]]s[i] \ne s[\pi[i-1]]s[i]=s[π[i−1]]，令 j=π[i−1] j = \pi[i-1] j=π[i−1]。若 s[i]≠s[j] s[i] \ne s[j] s[i]=s[j]，则令 j=π[j−1] j = \pi[j-1] j=π[j−1]，直到 j=0∨s[i]=s[j] j = 0 \vee s[i] = s[j] j=0∨s[i]=s[j] 为止，则

π[i]={0if s[i]≠s[j]j+1if s[i]=s[j]\begin{array}{c} \pi[i] = \begin{cases} 0 & if \ s[i] \ne s[j] \\ j + 1 & if \ s[i] = s[j] \end{cases} \end{array} π[i]={0j+1​if s[i]=s[j]if s[i]=s[j]​​

【注】计算字符串的前缀函数的思想和 KMP 算法中计算字符串失配数组的思想非常相似。

4. 应用

4.1 KMP

前缀函数可以用来实现 KMP 算法，思路为：拼接模式串 sss 和主串 ttt，得到 S=s+#+tS = s + \# + tS=s+#+t，#\## 为不在 sss 和 ttt 中出现的字符。设

m=∣s∣n=∣t∣\begin{array}{c} m = |s| \\ n = |t| \end{array} m=∣s∣n=∣t∣​

计算拼接后的字符串 SSS 的前缀函数，当出现 i>m∧π[i]=mi \gt m \wedge \pi[i] = mi>m∧π[i]=m 时，说明此时模式串匹配上了主串的子串 ti−2m⋯ti−m−1t_{i-2m} \cdots t_{i-m-1}ti−2m​⋯ti−m−1​。

整个算法时间复杂度为 O(n+m)O(n+m)O(n+m) 。

4.2 字符串周期 & border

根据上文中给出的性质，可以很容易求出字符串 sss 的字符串周期 & border。假设 ∣s∣=m|s| = m∣s∣=m，则可以在 O(m)O(m)O(m) 时间内求出 sss 的所有周期 & border。

4.3 统计每个前缀出现次数

  • 统计字符串 sss 的所有前缀子串在 sss 中出现的次数，m=∣s∣m = |s|m=∣s∣。
  1. 首先统计前缀数组值 π[i]\pi[i]π[i]，π[i]\pi[i]π[i] 表示字符串 s[0..i]s[0..i]s[0..i] 最长相等真前后缀长度，即说明前缀 s[0..π[i]−1]s[0..\pi[i]-1]s[0..π[i]−1] 在 s[0..i]s[0..i]s[0..i] 中出现了 1 次（不包括前缀本身）。
  2. 前缀数组值统计后，只统计出了每个前缀作为某个字符串 s[0..i]s[0..i]s[0..i] 的最长真后缀的出现次数，而没有统计非最长真后缀的出现次数，故根据 π\piπ 数组的性质统计非最长真后缀的出现次数。
  3. 加上每个前缀本身 1 次。
  • 统计字符串 sss 的所有前缀子串在 ttt 中出现的次数，m=∣s∣,n=∣t∣m = |s|, n = |t|m=∣s∣,n=∣t∣ 。拼接字符串 sss 和 ttt，使得 S=s+#+tS = s + \# + tS=s+#+t 。
  1. 首先统计前缀数组值 π[i],i>m\pi[i], i \gt mπ[i],i>m，π[i]\pi[i]π[i] 表示字符串 S[0..i]S[0..i]S[0..i] 最长相等真前后缀长度，即说明前缀 S[0..π[i]−1]S[0..\pi[i]-1]S[0..π[i]−1] 在 S[0..i]S[0..i]S[0..i] 中出现了 1 次（不包括前缀本身），易知最长真前后缀都不会包含界定符 #\##，故统计得到的只是字符串 ttt 中的。
  2. 前缀数组值统计后，只统计出了每个前缀作为某个字符串 S[0..i]S[0..i]S[0..i] 的最长真后缀的出现次数，而没有统计非最长真后缀的出现次数，故根据 π\piπ 数组的性质统计非最长真后缀的出现次数。

4.4 不同子串数目

给定字符串 sss，其长度 ∣s∣=m|s| = m∣s∣=m，计算 sss 中不同的子串的数目。

  • 设字符串 s[0..i]s[0..i]s[0..i] 的不同子串数目为 kkk，则向 s[0..i]s[0..i]s[0..i] 末尾添加一个字符后得到字符串 s[0..i+1]s[0..i+1]s[0..i+1]。显然 s[0..i+1]s[0..i+1]s[0..i+1] 的子串中可能会出现一些新的以 s[i+1]s[i+1]s[i+1] 结尾的子串。

  • 反转字符串 s[0..i+1]s[0..i+1]s[0..i+1] 得到字符串 ttt，则问题变成统计以 s[i+1]s[i+1]s[i+1] 开头且未在 ttt 的其他地方出现的前缀数目。

  • 设 ttt 的前缀函数的最大值为 πmax\pi_{max}πmax​，则最长的出现在 ttt 其他地方的前缀长度为 πmax\pi_{max}πmax​，故更短的前缀也一定出现了。

  • 因此，字符串 sss 新增一个末尾字符 s[i+1]s[i+1]s[i+1] 后新出现的子串的数目为 ∣s∣+1−πmax|s| + 1 - \pi_{max}∣s∣+1−πmax​。

【注】从头部添加、头部移除或尾部移除后计算不同子串的思想类似。

4.5 字符串压缩

  • 给定字符串 sss，其长度 ∣s∣=n|s| = n∣s∣=n，我们希望找到一个最短的字符串 ttt，使得 sss 为 ttt 的一份或多份拷贝的拼接表示。

  • 显然，我们只需要找到 ttt 的长度即可，该问题的答案即为长度为该值的 sss 的前缀。

根据上文的性质可知，如果计算出 sss 的前缀函数之后，sss 的最小周期为 k=n−π[n−1]k = n - \pi[n-1]k=n−π[n−1]。由字符串的周期的定义可知，最后字符串 sss 删去每段周期长度的字符串后，剩余的最后一段字符串长度不一定是 kkk。故如果 k∣nk | nk∣n，则 kkk 即是 ttt 的长度，否则不存在一个有效的压缩，即 ttt 的长度为 nnn 。
