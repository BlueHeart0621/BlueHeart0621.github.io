堆排序与优先队列

【注】参考自教材「算法导论」。

1. 堆

1.1 定义

（二叉）堆物理上是一个数组，逻辑上是一棵完全二叉树，树上的每一个结点对应数组中的一个元素。
设表示堆的数组 AAA，其包含两个属性：

  1. A.lengthA.lengthA.length 给出数组元素的个数。
  2. A.heapsizeA.heapsizeA.heapsize 给出有多少个堆元素存储在该数组中。

1.2 性质

从根结点编号为 1 开始，同一深度的结点自左往右、不同深度的结点自上往下，对堆上的结点编号（即逐层从左往右进行编号），结点编号对应数组 AAA 中元素的下标。

  • nnn 个元素的堆的高度为 ⌊lg⁡n⌋\lfloor \lg n \rfloor⌊lgn⌋ 。

证明
  因为堆是一棵完全二叉树，设堆高为 hhh，则

2h≤n≤2h+1−1⇒lg⁡(n+1)−1≤h≤lg⁡n⇒lg⁡n−1<lg⁡(n+1)−1≤h≤lg⁡n{\begin{array}{c} 2^h \leq n \leq 2^{h+1} - 1 \Rightarrow \lg (n+1) - 1 \leq h \leq \lg n \\ \Rightarrow \lg n - 1 \lt \lg (n+1) - 1 \leq h \leq \lg n \end{array}}2h≤n≤2h+1−1⇒lg(n+1)−1≤h≤lgn⇒lgn−1<lg(n+1)−1≤h≤lgn​

因为 hhh 为整数，故

h=⌊lg⁡n⌋{\begin{array}{c} h = \lfloor \lg n \rfloor \end{array}}h=⌊lgn⌋​

  • 非根结点 iii 的双亲结点的编号为

parent[i]=⌊i/2⌋{\begin{array}{c} parent[i] = \lfloor i/2 \rfloor \end{array}}parent[i]=⌊i/2⌋​

  • 非叶结点 iii 的左右孩子结点编号分别为

left[i]=2iright[i]=2i+1{\begin{array}{c} left[i] = 2i \\ right[i] = 2i + 1 \end{array}}left[i]=2iright[i]=2i+1​

  • 当用数组表示存储 nnn 个元素的堆时，叶结点编号分别是 ⌊n/2⌋+1,⌊n/2⌋+2,⋯ ,n \lfloor n/2 \rfloor + 1, \lfloor n/2 \rfloor + 2, \cdots, n ⌊n/2⌋+1,⌊n/2⌋+2,⋯,n 。

证明
  因为 nnn 是编号最大的结点，故 nnn 肯定是叶结点，且是编号最大的叶结点。故 nnn 的双亲 parent[n]=⌊n/2⌋parent[n] = \lfloor n/2 \rfloorparent[n]=⌊n/2⌋ 也是编号最大的非叶结点。故区间 (parent[n],n](parent[n], n](parent[n],n] 范围内的结点编号都是叶结点。即 ⌊n/2⌋+1,⌊n/2⌋+2,⋯ ,n \lfloor n/2 \rfloor + 1, \lfloor n/2 \rfloor + 2, \cdots, n ⌊n/2⌋+1,⌊n/2⌋+2,⋯,n 。

  • 对于包含 nnn 个元素的堆最多包含 ⌈n/2h+1⌉\lceil n/2^{h+1} \rceil⌈n/2h+1⌉ 个高度为 hhh 的结点。

证明
归纳法证明如下：

  • 对于高度为 0 的结点（即叶结点），因为堆是一棵完全二叉树，故叶结点个数 lll 满足 l0=⌊(n+1)/2⌋=⌈n/20+1⌉ l_0 = \lfloor (n+1)/2 \rfloor = \lceil n/2^{0 + 1} \rceil l0​=⌊(n+1)/2⌋=⌈n/20+1⌉
  • 假设对于高度为 hhh 的结点，满足结点数 lh=⌈n/2h+1⌉l_h = \lceil n/2^{h+1} \rceillh​=⌈n/2h+1⌉，由于堆是一棵完全二叉树，故对于高度为 h+1h+1h+1 的结点，有

lh+1=⌈lh/2⌉=⌈n/2(h+1)+1⌉{\begin{array}{c} l_{h+1} = \lceil l_h / 2 \rceil = \lceil n / 2^{(h+1)+1} \rceil \end{array}}lh+1​=⌈lh​/2⌉=⌈n/2(h+1)+1⌉​

堆可分为最大堆和最小堆，二者分别满足最大堆性质和最小堆性质。

  • 最大堆性质：除了根结点以外的所有结点 iii 满足

A[parent(i)]≥A[i]{\begin{array}{c} A[parent(i)] \geq A[i] \end{array}}A[parent(i)]≥A[i]​

  • 最小堆性质：除了根结点以外的所有结点 iii 满足

A[parent(i)]≤A[i]{\begin{array}{c} A[parent(i)] \leq A[i] \end{array}}A[parent(i)]≤A[i]​

2. 堆排序

堆排序是原址的、不稳定的。以下仅以最大堆为例。

2.1 maxHeapify

该过程用于维护最大堆性质。即根据输入的结点 iii 维护以 iii 为根的堆的最大堆性质。

由于堆高为 ⌊lg⁡n⌋\lfloor \lg n \rfloor⌊lgn⌋，故 maxHeapify 复杂度为 O(lg⁡n)O(\lg n)O(lgn) 。

2.2 buildMaxHeap

采用自底向上的方法，利用 maxHeapify 过程，将数组 A 构建为一棵最大堆。即从最小的非空子树（内部结点）开始，由于最小非空子树最少两个元素、最多三个元素，故 maxHeapify 后一定是一棵最大堆，然后逐步向上对更大的子树递归利用 maxHeapify 构建最大堆。
根据上文堆的性质可知，堆的叶结点编号分别是 ⌊n/2⌋+1,⌊n/2⌋+2,⋯ ,n \lfloor n/2 \rfloor + 1, \lfloor n/2 \rfloor + 2, \cdots, n ⌊n/2⌋+1,⌊n/2⌋+2,⋯,n ，故非叶结点的编号为 1,2,⋯ ,⌊n/2⌋ 1,2, \cdots, \lfloor n/2 \rfloor 1,2,⋯,⌊n/2⌋ 。

因为在高度为 hhh 的树上执行 maxHeapify 过程的复杂度为 O(h)O(h)O(h)，而由上文堆的性质可知，堆中高度为 hhh 的结点数为 ⌈n/2h+1⌉\lceil n/2^{h+1} \rceil⌈n/2h+1⌉，堆的高度为 lg⁡n\lg nlgn，故 buildMaxHeap 的复杂度为

∑h=0⌊lg⁡n⌋⌈n/2h+1⌉O(h)=O(n){\begin{array}{c} \sum^{\lfloor \lg n \rfloor}_{h = 0}\lceil n/2^{h+1} \rceil O(h) = O(n) \end{array}}∑h=0⌊lgn⌋​⌈n/2h+1⌉O(h)=O(n)​

2.3 heapSort

构建好最大堆后，值最大的元素在 A[1]，通过将 A[1] 与 A[A.heapsize] 交换后，然后缩小 A.heapsize，并使用 maxHeapify 过程维护最大堆性质。如此往复，直到 A.heapsize = 2 时，数组 A 变为有序。

heapSort 的复杂度为 O(n+⌊lg⁡n⌋+⌊lg⁡(n−1)⌋+⋯+⌊lg⁡1⌋)=O(n+lg⁡n!)=O(nlg⁡n)O(n + \lfloor \lg n \rfloor + \lfloor \lg (n-1) \rfloor + \cdots + \lfloor \lg 1 \rfloor) = O(n + \lg n!) = O(n\lg n)O(n+⌊lgn⌋+⌊lg(n−1)⌋+⋯+⌊lg1⌋)=O(n+lgn!)=O(nlgn) 。

3. 优先队列

以下优先队列以最大堆为底层实现。

3.1 maxHeapifyUp

该过程用于从当前结点向根结点的路径上维护最大堆性质。

由于堆高为 ⌊lg⁡n⌋\lfloor \lg n \rfloor⌊lgn⌋，故 maxHeapifyUp 复杂度为 O(lg⁡n)O(\lg n)O(lgn) 。

3.2 maxHeapifyDown

该过程用于从当前结点向叶结点的路径上维护最大堆性质。

由于堆高为 ⌊lg⁡n⌋\lfloor \lg n \rfloor⌊lgn⌋，故 maxHeapifyDown 复杂度为 O(lg⁡n)O(\lg n)O(lgn) 。
