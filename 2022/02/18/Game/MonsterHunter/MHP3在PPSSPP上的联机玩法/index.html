MHP3在PPSSPP上的联机玩法

1. 简介

MHP3 是 CAPCOM 公司针对 PSP 平台开发的一款 ARPG 游戏，于 2010 年 12 月 1 日发行。MHP3 也是我首个玩的怪物猎人游戏，个人觉得这个游戏做得非常不错，虽然它的画质在现在看来太低，但其给人的游戏体验丝毫不差。为了重温经典，特别是多人联机对战（以前我都是玩的单机），我选择了 PPSSPP 这款模拟器。PPSSPP 开源且跨平台，支持联机作战，游戏进度数据也容易保存和迁移。

2. 设置

【注】以下配置中，除明确说明的外，均需对所有联机设备进行设置。

2.1 初始配置

要实现 MHP3 的联机，首先需要对每个设备上的 PPSSPP 软件进行配置，打开 「Settings」菜单：

  • 关闭「System」子菜单中的「Fast memory」选项；
  • 打开「Networking」子菜单中的「Enable networking/WLAN」选项；
  • 将「Networking」子菜单中的「WLAN channel」设置为 auto；
  • 将「Networking」子菜单中的「Port offset」设置为相同的值，比如都设置为 10000。

2.2 联网配置

实现 PPSSPP 上的 MHP3 游戏的联网，需要让所有设备上的 PPSSPP 都连接到同一个 PRO Adhoc 服务器。这个 PRO Adhoc 服务器可以使用公开的，也可以使用 PPSSPP 内置的自建。

2.2.1 公共 PRO Adhoc 服务器

目前公开的 PRO Adhoc 服务器主要有两个：

  • https://myneighborsushicat.com/
  • https://socom.cc

如果使用公共的 PRO Adhoc 服务器，则只需要将公共服务器的域名填入「Networking」子菜单中的「Change PRO ad hoc server IP address」项中即可。或者直接打开「Change PRO ad hoc server IP address」项，点击「Toggle list」选项也可以找到这两个公共服务器的域名，直接选中即可。

2.2.2 内置 PRO Adhoc 服务器

如果不想使用公共的服务器，可以使用 PPSSPP 内置的 PRO Adhoc 服务器。多人联机，只需要一台设备开启内置的 PRO Adhoc 服务器，其他的设备连接该服务器即可。

  • 开启内置的 PRO Adhoc 服务器的设备需要开启「Networking」子菜单中「Enable built-in PRO ad hoc server」选项。

  • 然后查看开启了 PRO Adhoc 服务器的设备的 IP 地址，Windows 平台使用 ipconfig 命令查看，Linux 平台使用 ip a 命令查看。

  • 最后将查询到的 PRO Adhoc 服务器的所在设备的 IP 地址，「Networking」子菜单中的「Change PRO ad hoc server IP address」项中即可。

【注】如果开启了内置 PRO Adhoc 服务器的设备也需要连机，则其同样需要配置「Change PRO ad hoc server IP address」项。
