2.1 Qv2ray

Qv2ray 兼容性非常好，不仅原生支持 SS、V2ray，还能够通过插件支持诸如 Trojan、Trojan-go 等。以下是 Qv2ray 可扩展的插件列表，只需下载相应的插件库文件到 Qv2ray 的插件目录下即可。

打开 Qv2ray，依次点击「Plugins」->「Open Local Plugin Folder」即可找到 Qv2ray 的插件目录。

支持的协议 插件下载
SSR QvPlugin-SSR
Trojan QvPlugin-Trojan
Trojan-go QvPlugin-Trojan-go
NaiveProxy QvPlugin-NaiveProxy

添加完插件库文件重启 Qv2ray 即可使用。

2.2 Clash

Clash 是目前仅存的几款好用的代理工具（Qv2ray 已不再更新维护了～）。Clash 本身是一款 CLI 后端工具，其详细配置可以参考官方文档。对于 GUI 图形化客户端，也已经有大佬写出来了，Clash 目前的 GUI 客户端有：

应用 支持平台
Clash for Windows Windows/macOS/Linux
Clash for Windows Chinese Windows/macOS/Linux
Clashy Windows/macOS/Ubuntu
ClashWeb Windows
Clash for Flutter Windows/macOS/Linux
Clash for Android Android
