实用工具合集

1. When2Meet

When2Meet 是一个用来规划集体空闲时间段的实时网页工具，使用它可以很方便地统计大家的空闲时间段。

2. PDF24

PDF24 提供了很多关于 PDF 的工具，诸如「PDF 分割」、「PDF 合并」、「PDF 转图像」、「图像转 PDF」等。它不仅提供网页版工具，还提供可下载到本地的离线工具。

3. FlameShot

FlameShot 是一款优秀的跨平台截屏工具，适配了 Windows、Linux 等主流操作系统。

4. Free Download Manager

Free Download Manager 是一款优秀的跨平台下载加速器，适配了 Windows、Linux 等主流操作系统。

5. DrawIO

DrawIO 是一款优秀的跨平台绘图工具，提供绘制流程图、类图、关系图等各种图，同时支持 LaTeX 数学公式，并且支持多种格式导出，诸如 JPG、PNG、SVG 等。

6. PicGo

PicGo 是一款用于快速上传图片并获取图片 URL 链接的工具，适配了 Windows、Linux 等主流操作系统。

7. BestTrace

BestTrace 是一款多平台的路由跟踪测试工具，支持显示 IP 归属地信息。

8. PDFtk

PDFtk 简直是命令行用户的福音，它提供 PDF 文档分割、合并、加解密、翻转页面、等诸多功能，而且它也是多平台的一款软件。其使用简单，具体例子可以参考官方示例。

9. Gimp

Gimp 是一款强大的跨平台开源图像编辑软件，可以很好地作为 PhotoShop 的替代品。此外，Gimp 还有很多第三方插件，更加丰富了 Gimp 的功能，而且只要你想，你也能通过编程定制自己想要的功能。

10. Inkscape

Inkscape 是一款强大的跨平台开源矢量设计软件，可以很好地作为 Illustrator 的替代品。此外，Inkscape 官网上还给出了丰富的教程，同时还设有论坛供我们学习交流。

11. OpenShot

OpenShot 是一款强大的跨平台开源视频编辑软件，可以很好地作为 Premiere 的替代品。而且，OpenShot 非常简单易用，还有翔实的文档和视频教程，开发者实在太有爱了，赞～。

12. ImageMagick

ImageMagick 是一款非常强大的图片处理工具集合，它提供了各种图像处理功能，ImageMagick 在一般 Linux 发行版上都是预装的软件。

ImageMagick 的常用操作脚本库：Fred's ImageMagick Scripts

13. Poppler

Poppler 是一个基于 Xpdf 的 PDF 渲染库，它提供了很多 PDF 编辑工具，诸如 PDF 转图片、从 PDF 中提取文件、分割 PDF 文件、合并 PDF 文件等。

14. Audacity

Audacity 是一款强大的跨平台开源音频编辑软件，一定程度上能够作为 Audition 的替代品。不过，相较于 Audition，Audacity 在专业功能上还是有所欠缺。

15. Aria2

Aria2 是一个跨平台的轻量级、支持多种协议的命令行下载工具。它支持的协议包括 HTTP(S)、FTP、BitTorrent、Metalink 等。aria2 工具可以并发地进行下载，同时还支持分片与续传，自动替换慢的线程等。

  • Aria2 教程

16. Motrix

Motrix 是一款基于 Aria2 全能的下载工具，支持下载 HTTP、FTP、BT、磁力链等资源。

17. Zsh

Zsh 是一款专为交互使用而设计的 Shell，其基于 Bourne Shell 作出了大量改进，同时加入了 bash、ksh 和 tcsh 的一些功能。此外，Zsh 还拥有大量的插件，提供了丰富的实用功能。

18. Tmux

Tmux 是一款好用的 Shell 终端复用器，它可以在一个屏幕上创建、访问和控制多个终端。此外，Tmux 还可以从屏幕上分离并继续在后台运行，然后再重新连接。

19. AppImageKit

AppImageKit 是一个非常有用的 Linux 软件打包工具，它能将 Linux 软件打包成不依赖额外库的 AppImage 格式。AppImage 格式的软件无需安装，也不需要解压之类的，可以直接运行在所有主流 Linux 发行版上。

20. Mathpix-Snipping-Tool

Mathpix-Snipping-Tool 是一款非常实用的 OCR 工具，它不仅可以识别文字，更重要的是它可以识别数学公式并自动转为 Word/Latex/Markdown 公式，用户只需复制粘贴即可，有了它妈妈再也不用担心我打公式打到手麻了。我直接吹爆～简直是水写博客的神器！唯一的缺点是每个月的识别次数有限，毕竟白嫖别人的服务。

21. Wolframalpha

Wolframalpha 是一个提供各种数学功能的在线网站，它提供包括积分计算、图形计算、几何计算、可视化等强大功能。此外，Wolframalpha 还提供 Wolfram 语言，其是脱胎于 Mathematica 的基于知识的独特符号式语言。

22. Evince

Evince 是 GNOME 桌面环境下的一个 PDF、PostScript、DjVu、TIFF 与 DVI 文档查看工具。其和一般的 PDF 查看器相比 Evince 的好处在于，它有矢量打印的功能，即对于一个矢量 PDF，Evince 能够以矢量格式将其虚拟打印出来，而一般的 PDF 阅读器都是以标量图的形式打印 PDF。

23. ITellYou

ITellYou 是一个原版软件信息收录站点，其主要收录 Windows 下的系统和应用。不过这个站点的作者似乎又开了一个新站点：next.itellyou.cn，其上面也开始提供 Linux 和 Mac 系统，后续估计也会越做越好。

24. ZeroTier

ZeroTier 是一款跨平台的虚拟局域网构建工具，可以很方便地用来创建自己的虚拟局域网，实现内网穿透。其客户端源代码见官访仓库：ZeroTierOne。

25. SageMath

SageMath 是一款开源免费的数学软件，其支持代数、几何、数论、密码学、数值计算和相关领域的研究和教学。SageMath 的目标是为「4M」，即 Maple、Mathematica、Magma 和 Matlab 提供一个可行的、免费的、自由的替代品。

26. PKGS.ORG

PKGS.ORG 是一个提供 Unix/Linux 最新软件包镜像下载的网站，诸多 Unix/Linux 发行版的软件都能在上面找到。
