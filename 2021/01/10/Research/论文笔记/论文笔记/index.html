1. Learning Continuous Image Representation with Local Implicit Image Function

  • 首先利用 SOTA 的超分网络中的 Encoder 部分得到 Feature Map，然后利用 LIIF（本质为 MLP）将 Feature Map 超分到任意分辨率。
  • Local Implicit Image Function（LIIF）：
  1. 将 Feature Map 映射到横纵坐标区间均为 [-1,1] 的区域内，对于每个坐标点先将其周围 3x3=9 个Feature Vectors 拼接在一起替换该坐标点的 Feature Vector（充分利用局部信息）；
  2. 然后计算要超分的图像映射到 [-1,1]x[-1,1] 区间的坐标，每个坐标点分别取其左上、右上、左下、右下距离其最近的 Feature Vector 作为 LIIF 的输入，计算结果的加权求和，距离坐标点越近的 Feature Vector 对应的权值越大。
  3. 此外，还加入每个像素点负责的区域大小和坐标点拼接在一起作为 LIIF 的输入。

5. NeRF: Representing Scenes as Neural Radiance Fields for View Synthesis

  • 将 3D 场景表示为 5D 的函数 fff，其中 (x,y,z)(x,y,z)(x,y,z) 为空间坐标，(σ,ϕ)(\sigma,\phi)(σ,ϕ) 为观测角度；函数输出为该位置观测到的 RGB 颜色以及 Density。

  • 基于 fff 的预测输出，利用传统的 Rendering 技巧来生成图像。
